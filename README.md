# LinkRSP

**Link Reshuffling & Survival Protocol（劳动力再平衡与生存韧性协议）**

本仓库承载 LinkRSP 的规范、评估报告与后续工程化材料；与 [HDGP（厚德归朴）](https://github.com/HumanDignityGuardian/HDGP-Core) 在治理与审计层对接，作为其上的**劳务与社会韧性协议**场景延伸。

## 上层指导（本地）

协议与引擎的伦理、规则引擎与集成方式以本地工作区为准：

| 路径 | 说明 |
|------|------|
| `D:\HDGP\HDGP-Protocol` | HDGP 主工作区：规范、Engine API、集成单页、伦理基线等 |

详见 [`docs/parent-hdgp.md`](docs/parent-hdgp.md)。
